Wallpaper

A Stunning Wallpaper Changer

Download as .zip Download as .tar.gz View on GitHub

Wallpaper is a wallpaper changer for GTK3 DE's.

Wallpaper can change wallpapers in Gnome Cinnamon Pantheon and anything else that uses the same gnome like session keys.

Currently to get Wallpaper you have to clone from github like so.

$ git clone https://github.com/infektedpc/Wallpaper.git

You Need the gambas 3.0 runtime or higher in ubuntu and derivatives you can use the "Ubuntu Gambas Maintainers Team repo

$ sudo add-apt-repository ppa:gambas-team/gambas3
$ sudo apt-get update
$ sudo apt-get install gambas3-runtime
or if you want to develop on the Program
$ sudo apt-get install gambas3
